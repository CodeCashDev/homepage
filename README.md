# CodeTas Homepage

CodeTasのコーポレートランディングページです。Web・モバイルアプリの開発および運用支援を紹介する、GitHub Pages向けの静的サイトです。

## ローカル確認

依存パッケージはありません。`index.html`をブラウザで開くか、任意のローカルHTTPサーバーを使用してください。

```bash
python3 -m http.server 8000
```

`http://localhost:8000` にアクセスします。

## 公開前の確認

- `index.html`の問い合わせメールアドレスを実際のアドレスに変更する
- GitHubの `Settings > Pages` でデプロイ元を `main` / `root` に設定する
- 独自ドメインを利用する場合はPages設定とDNSレコードを設定する
